## Description

- target: 취업준비생, 이직준비자

- 데이터 수집, 저장, 분석, 화면구현

- 취준생들이 회사를 고르는 기준

        1. 연봉(업계 평근 대비 연봉, 연봉 인상률 등)
    
        2. 복지(상여금, 식비지원, 경조사지원 등)
        
        3. 회사비전(매출, 직원수, 순이익, 업계 점유율, 발전 가능성 등)
    
        4. 조직문화(워라벨, 인사평가 시스템 등)

        5. 근무지(출퇴근 거리, 역세권)

- 기존(경쟁) 웹 또는 앱 장단점 분석

        1. 유용성: 구인글, 원하는 정보

        2. 편의성: 검색필터설정, 로그인 편의성, 이력서 관리

        3. 접근성: 모바일 앱, 웹사이트

        4. 맞춤성: 개인화
